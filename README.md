# 🧠 Multi-Task Computer Vision System

A comprehensive **AI-powered Multi-Task Computer Vision Application** developed using **Python, Streamlit, TensorFlow/Keras, YOLOv8, OpenCV, and Machine Learning**. This project combines multiple computer vision tasks into a single interactive web application.

---

## 📌 Project Overview

This application demonstrates the practical implementation of Deep Learning and Computer Vision techniques through multiple real-world image analysis tasks. The project provides an intuitive interface where users can upload images or use a webcam (when running locally) to perform different AI-powered detections.

---

## ✨ Features

* 👤 Age Detection
* 🚻 Gender Detection
* 💇 Long Hair Detection
* 👴 Senior Citizen Detection
* 🤟 Sign Language Recognition
* 🚗 Car Colour Detection

# ⚙️ Installation

## Step 1 : Clone the Repository

```bash
git clone https://github.com/NIKHIL-TEJA341/ageandgenderdetection.git
```
---

## Step 2 : Move into the Project Directory

```bash
cd project-name
```

---

## Step 3 : Install Required Libraries

```bash
pip install -r requirements.txt
```
---

# ▶️ Run the Application

```bash
python -m streamlit run app.py
```

The application will automatically open in your default browser.

If it doesn't, open:

```
http://localhost:8501
```
---

# 🚀 How to Use

1. Launch the Streamlit application.
2. Select the desired module from the sidebar.
3. Upload an image (or use the webcam when running locally if supported).
4. Wait for the trined model to process the image.
5. View the prediction results.
